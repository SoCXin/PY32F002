================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了LPTIM中断唤醒stop模式，500ms唤醒一次

================================================================================
测试环境：
测试用板：PY32F002A_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行
2. 按下按键进入stop模式；
3. 500ms后唤醒stop模式并循环进入stop模式再唤醒；
4. LED保持1Hz频率闪烁；

================================================================================
注意事项：
每次唤醒后再次进入Stop模式前,必须增加75us(至少两个LSI时钟)延时,用于LSI和APB时钟
同步

================================================================================
