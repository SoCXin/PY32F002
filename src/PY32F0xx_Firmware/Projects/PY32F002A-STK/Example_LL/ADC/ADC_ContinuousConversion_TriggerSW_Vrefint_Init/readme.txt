================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC模块的VCC采样功能，通过采样VREFINT,反推VCC电压。

================================================================================
测试环境：
测试用板：PY32F002A_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序
2. 通过串口查看打印的VCC电压值

================================================================================
注意事项：
STK板        USB转TTL模块
PA02(TX) --> RX
PA03(RX) --> TX
GND      --> GND

================================================================================
