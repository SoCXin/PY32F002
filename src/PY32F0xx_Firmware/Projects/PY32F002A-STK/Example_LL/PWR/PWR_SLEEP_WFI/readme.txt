================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了通过 WFI(wait for interrupt)指令进入sleep模式，使用GPIO中断唤醒

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. LED闪烁一次后进入sleep模式；
3. 在PA06上产生一个上升沿，触发GPIO中断，在中断中点亮LED，程序退出sleep模式；
4. LED指示灯每隔200ms进行翻转；
