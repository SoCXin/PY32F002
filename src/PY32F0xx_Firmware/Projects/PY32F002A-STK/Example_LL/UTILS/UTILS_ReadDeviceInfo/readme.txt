================================================================================
                                样例使用说明
================================================================================
功能描述：
本样例主要读取DBGMCU->IDCODE寄存器和UID的值。
UID Word0表示LotNumber, Word1表示WaferNumber, Word2表示X和Y的坐标。

================================================================================
测试环境：
测试用板：PY32F002A_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 仿真模式下在Watch窗口查看全局变量aShowDeviceID, aShowRevisionID,
   aShowCoordinate, aShowWaferNumber, aShowLotNumber的值。
