================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了GPIO外部中断功能，PB2引脚上的每一个下降沿都会产生中断，中断函数中
LED灯会翻转一次

================================================================================
测试环境：
测试用板：PY32F002A_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 每按一次用户按键，观察到LED翻转一次；

================================================================================
注意事项：
用户按键所对应的引脚为PB2

================================================================================
