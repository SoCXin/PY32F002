================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了sleepp模式下，GPIO外部中断唤醒功能

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 按下复位按键，程序进入sleep模式，串口打印：
   SLEEP MODE!
3. 在PB2上产生一个下降沿，程序退出sleep模式，并且串口打印：
   WAKEUP OK!
4. 按键后，LED状态会进行翻转，每按一次，翻转一次

================================================================================
注意事项：
MCU与串口调试助手连线:
RX ---> PA2
TX ---> PA3
波特率：115200bps

演示此样例功能时需要断开swd连接线，因为默认情况下，仿真器会把DBGMCU_CR.DBG_STOP置位
注意关闭systick中断，防止该中断唤醒MCU：
