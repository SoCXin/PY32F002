================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了通过PA12引脚唤醒MCU的功能，下载程序后，MCU会进入STOP模式，按下用户键
MCU会退出STOP模式，然后LED会以500ms的间隔进行翻转

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 进入STOP后，按下用户按键，LED会以500ms的间隔进行翻转
