================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例是对高级定时器功能“六步PWM的产生”的演示，通过systick中断作为COM commutation
事件的触发源，实现（无刷电机的）换向

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译并下载程序；
2. 刹车输入（PA06）接高电平；
3. 用示波器或逻辑分析仪接以下六个通道，可监测到上述描述波形；
4. 刹车输入（PA06）接高电平，触发刹车，可观测到上述波形没有了，并输出空闲电平（空
闲电平有寄存器配置决定）

================================================================================
注意事项：
下表是换向步骤，比如第一步中的CH1和CH3N为1，即设置打开这两个通道的PWM输出
