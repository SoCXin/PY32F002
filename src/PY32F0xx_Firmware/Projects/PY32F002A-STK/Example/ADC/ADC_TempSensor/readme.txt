================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC模块的温度传感器功能，通过仿真器，在中断根据观测采样值
（aADCxConvertedData），计算出当前的温度。

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 根据实际VCC电压修改参数Vcc_Power
2. 编译下载程序到MCU，并运行；
3. 串口每隔500ms输出当前的温度；
================================================================================
注意事项：
注意事项：
MCU端串口打印配置：
波特率：115200
RX --> PA2
TX --> PA3
================================================================================
