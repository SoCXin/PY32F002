================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC模块的VCC采样功能，通过采样VREFINT,反推VCC电压

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载仿真,
2. 通过观察T_VCC里的值,是否和VCC电压一致,
3. 建议VCC供电调整要覆盖常用电压3.3V和5.0V
