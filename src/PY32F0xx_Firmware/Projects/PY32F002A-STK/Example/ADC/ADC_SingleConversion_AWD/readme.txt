================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC的模拟看门狗功能.

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 给AN0(PA0)或AN1(PA1)大于供电电压的1/8
2. 运行程序,则发现程序会进入看门狗中断
