================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了LPTIM中断唤醒stop模式，200ms唤醒一次。

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1.编译下载程序到MCU，并运行；
2.按下按键进入stop模式；

结果:
LED 2.5HZ闪烁表示运行正常
================================================================================
注意事项：
每次唤醒后再次进入Stop模式前,必须增加75us(至少两个LSI时钟)延时,用于LSI和APB时钟同步

================================================================================
