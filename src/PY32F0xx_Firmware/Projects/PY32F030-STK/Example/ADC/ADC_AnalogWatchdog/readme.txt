================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC的模拟看门狗功能，当开启看门狗的通道的电压值不在设定的上下限中，会
进入看门狗中断。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. PA0和PA1的电压值都小于0.4125V(供电电压为3.3V),不会进入看门狗中断，LED灯不亮；
3. PA0或PA1的电压值大于0.4125V(供电电压为3.3V)，进入看门狗中断，LED亮。
