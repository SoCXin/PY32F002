================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了TIM1和TIM3同步触发（复位触发模式）功能，TIM1作为主机，配置复位TRGO
输出(TIM1_CR2.MMS=000)，TIM3作为从机并且使能事件更新中断，配置收到主机的TRGO信
号后复位(TIM3_SMCR.SMS=100)。循环打印TIM3计数值。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序并运行；
2. 查看串口打印数值，TIM3计数值永远小于TIM1重装载值；

================================================================================
注意事项：
PA2 -> TX
