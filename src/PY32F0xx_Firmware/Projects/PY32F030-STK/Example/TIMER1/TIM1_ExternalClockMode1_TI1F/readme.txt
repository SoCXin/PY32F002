================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了TIM1的外部时钟模式1功能，选择TI1FD(PA8)引脚作为外部时钟输入源，并使能
更新中断，在中断中翻转LED灯

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. PA8引脚，输入外部时钟；
3. 可观察到MCU从PA8引脚上输入时钟信号，CNT计数器计到800时，产生更新事件，LED翻转一次
