================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例实现了定时器的互补输出功能，三组互补共六路pwm输出，此样例没有实现死区功能
CH1   ->  PA8
CH1N  ->  PA7
CH2   ->  PA9
CH2N  ->  PB0
CH3   ->  PA10
CH3N  ->  PB1

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 下载并运行程序
2. 通过逻辑分析仪捕捉上述六通道的电平，判断pwm输出信号是否正确
