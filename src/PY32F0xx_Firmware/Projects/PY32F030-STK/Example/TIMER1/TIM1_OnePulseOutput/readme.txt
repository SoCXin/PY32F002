================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了TIM1的单脉冲模式，CH2(PA09)引脚上的上升沿，触发计数器开始计数，当计数
值与CCR1匹配时，CH1(PA08)输出高电平，直到计数器溢出，CH1再次输出低电平，计数器溢
出后，定时器停止工作，
本例程脉冲宽度计算 (TIM1_ARR-TI1_CCR1)/CLK=（65535-16383）/32000000=1.536ms
================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 在PA09上产生一个上升沿；
3. 通过示波器监测PA09和PA08引脚，可监测到波形;

================================================================================

时钟主频为32M
