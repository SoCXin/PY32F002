================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了GPIO输出模式，配置LED引脚(PA11)为数字输出模式，并且每隔250ms翻转一次
LED引脚电平，运行程序，可以看到LED灯以2Hz的频率闪烁。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 可观察到STK板上的LED以2Hz的频率闪烁。
