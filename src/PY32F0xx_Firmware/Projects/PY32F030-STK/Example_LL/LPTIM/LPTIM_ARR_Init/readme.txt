================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了LPTIM单次计数功能，配置LPTIM计数周期为0.5s，在ARR中断回调函数中翻转LED
灯，并再次启动单次计数模式。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行
2. LED灯以1Hz的频率闪烁
