================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示通过RTC秒中断从STOP模式下唤醒，唤醒后，小灯处于闪烁状态；否则处于熄灭
状态。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20

================================================================================
使用步骤：
1. 编译并下载程序到MCU；
2. 等待LED灯亮后，按下用户按键，LED灯关闭，进入STOP模式
3. 通过秒中断唤醒MCU后，LED灯闪烁
