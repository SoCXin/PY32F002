================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了DMA从SRAM到SRAM传输数据的功能(SRAM和外设之间传输的样例请参考相关外设
样例工程)。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. LED灯常亮，表明DMA传输完成并且数据正确；
3. LED灯闪烁，表明DMA传输出错或者DMA的数据传输的不正确。
