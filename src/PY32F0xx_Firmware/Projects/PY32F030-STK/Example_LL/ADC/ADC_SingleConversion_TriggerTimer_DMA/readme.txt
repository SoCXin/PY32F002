================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC采集通过TIM触发采集并通过DMA传输的模式，PA4为模拟输入，每隔1s会从
串口打印当前的电压值。


================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 定时器每隔1s触发一次ADC采样；
3. 每完成一次采样，串口打印一次PA4上的采样值。

================================================================================
注意事项：
STK板        USB转TTL模块
PA02(TX) --> RX
PA03(RX) --> TX
GND      --> GND

波特率:115200

================================================================================
