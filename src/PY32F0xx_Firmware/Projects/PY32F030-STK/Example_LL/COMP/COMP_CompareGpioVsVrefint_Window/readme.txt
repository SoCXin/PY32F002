================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了COMP比较器的window功能，比较器2的Plus端用比较器1的IO3(PA1)作为输入，
VREFINT作为比较器负端输入，当PA1的电压值大于1.3V时,LED灯亮，小于1.1V时,LED灯灭。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 在PA01上输入大于1.3V的电压，LED灯亮；
3. 在PA01上输入小于1.1V的电压，LED灯灭。
