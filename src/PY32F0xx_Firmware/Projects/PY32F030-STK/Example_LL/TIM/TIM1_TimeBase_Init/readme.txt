================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了TIM1的重装载功能，配置计数周期为1s，在ARR中断回调函数中翻转LED。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行
2. LED每过1s翻转一次
