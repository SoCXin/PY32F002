================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了使用TIM1 PWM2模式输出三路频率为10Hz占空比分别为25%、50%、75%的PWM波形。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行
2. 使用逻辑分析仪观察引脚PA8/PA9/PA10的波形

================================================================================
注意事项：
PA8  -> CH1  占空比：75%
PA9  -> CH2  占空比：50%
PA10 -> CH3  占空比：25%

================================================================================
