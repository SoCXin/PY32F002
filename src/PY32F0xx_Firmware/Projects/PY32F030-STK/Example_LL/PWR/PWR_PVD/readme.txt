================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了PVD电压检测功能，样例中配置PB07引脚的电压与VREF(1.2v)进行比较，当
PB07引脚的电压高于VREF时,LED灯灭，当低于VREF时，LED灯亮。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行，LED灯亮；
2. PB07输入0V时，进入PVD中断，LED灯亮，PB07悬空时（内部上拉相当于接入3.3V）LED灯
灭；
3. 再次输入0V，LED灯亮。

================================================================================
注意事项：
本例程中PA11引脚默认上拉，因此默认PVDO状态为0。

================================================================================
