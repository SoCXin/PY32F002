================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例配置系统时钟为LSE，并通过MCO（PA08）引脚输出，注意系统时钟切换为LSE之前，要
求把systick中断关闭掉，因为systick中断默认是1ms一次中断，由于LSE时钟频率过低，
systick中断会导致程序无法正常运行。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 按下用户按键；
3. 用示波器监测PA08引脚上的频率，观察是否和配置的LSE频率一致；


================================================================================
注意事项：
1. 样例中默认启动的系统时钟是HSI，只有按下用户按键后，系统时钟才会切换到LSE。

================================================================================
