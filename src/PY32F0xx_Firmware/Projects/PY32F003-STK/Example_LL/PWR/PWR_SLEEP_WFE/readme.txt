================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了在sleep模式下，使用GPIO事件唤醒。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. LED灯亮，按下按键，LED灯灭，进入sleep模式；
3. 在PA06上产生一个下降沿，程序退出sleep模式；
4. LED以200ms的间隔进行翻转。
