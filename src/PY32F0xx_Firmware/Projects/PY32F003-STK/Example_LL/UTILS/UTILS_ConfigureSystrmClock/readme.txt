================================================================================
                                样例使用说明
================================================================================
功能描述：
本样例主要演示如何配置SYSCLK(系统时钟), HCLK(AHB时钟), PCLK(APB时钟)。
通过MCO输出系统时钟8MHz。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20

================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 通过逻辑分析仪或示波器可观测到端口PA1输出频率为8MHz的波形；
2. LED灯按照1s的频率闪烁。
