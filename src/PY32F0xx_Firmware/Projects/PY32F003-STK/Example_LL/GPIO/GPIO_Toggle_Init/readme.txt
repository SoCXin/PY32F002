================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了GPIO输出模式，配置LED引脚为数字输出模式，并且每隔100ms翻转一次LED引
脚电平，运行程序，可以看到LED灯闪烁。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. STK板上的LED闪烁。
