================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC模块的温度传感器功能，程序下载后，串口每隔200ms打印一次当前检测的
温度值和对应的采样值。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 串口每隔200ms输出当前的温度和对应的采样值。

================================================================================
注意事项：
STK板        USB转TTL模块
PA02(TX) --> RX
PA03(RX) --> TX
GND      --> GND


================================================================================
