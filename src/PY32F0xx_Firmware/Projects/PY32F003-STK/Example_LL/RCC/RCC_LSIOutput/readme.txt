================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了时钟输出功能，可输出LSI波形

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 进入仿真模式
3. 运行，可由示波器观察到PA01输出LSI波形(32.768KHz)
