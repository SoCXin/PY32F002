================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了时钟切换，由LSI切换至HSE

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 进入仿真模式
3. 单步执行，可由示波器观察到PA01输出的时钟波形切换（由32.768KHz切换至24MHz）
