================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了使用TIM1输出频率为10Hz占空比为50%的PWM波形以及它的互补信号。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行
2. 使用逻辑分析仪观察引脚PA3/PA7的波形

================================================================================
注意事项：
PA3  -> CH1  占空比：50%
PA7  -> CH1N

================================================================================
