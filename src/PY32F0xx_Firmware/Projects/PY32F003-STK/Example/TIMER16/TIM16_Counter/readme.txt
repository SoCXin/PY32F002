================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了在TIM16中基本计数功能，并使能了更新中断，每次重装ARR值时会产生一次
更新中断，并在中断中翻转LED灯，LED灯会以5Hz的频率进行翻转。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序并运行；
2. LED灯闪烁
