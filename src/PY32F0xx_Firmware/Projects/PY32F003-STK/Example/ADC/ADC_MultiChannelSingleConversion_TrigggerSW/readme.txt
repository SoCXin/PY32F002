================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC单次轮询(Polling)的方式采样AN0,AN1,AN4通道,并通过串口打印出来。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 串口循环打印通道0，1，4的电压。

================================================================================
注意事项：
STK板        USB转TTL模块
PA02(TX) --> RX
PA03(RX) --> TX
GND      --> GND

波特率:115200

================================================================================
