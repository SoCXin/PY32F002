================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了WWDG的提前唤醒中断功能，看门狗计数器向下计数到0x40时产生中断，中断中
喂狗，可以确保看门狗不会复位。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序并运行；
2. 程序在看门狗提前唤醒终端中喂狗，LED以1hz在闪烁。
