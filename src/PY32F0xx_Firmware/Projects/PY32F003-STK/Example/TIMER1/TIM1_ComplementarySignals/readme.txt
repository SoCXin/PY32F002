================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例实现了定时器的互补输出功能，一组互补共两路pwm输出，此样例没有实现死区功能
CH1   ->  PA3
CH1N  ->  PA0


================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 下载并运行程序
2. 通过逻辑分析仪捕捉上述两通道的电平，判断pwm输出信号是否正确
